# 유튜브 클론코딩
## 개요
유튜브 Open API를 통해 추천 동영상 목록, 동영상 검색, 동영상 재생 기능 구현 및 다크모드 기능 구현
## 사용 기술
* Youtube Open API
* HTML, JavaScript
* React, Tailwind CSS, React Router, React Query, Axios, React-icons
## 실행 방법
```
npm run start
```
